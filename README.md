# AVL Tree Visualizer 🌳

A React-based interactive AVL Tree Visualizer that demonstrates insertion operations with automatic balancing using rotations.

## 🚀 Features

- Insert nodes into an AVL Tree
- Self-balancing using AVL rotations (LL, RR, LR, RL)
- Real-time tree visualization using SVG
- Clean and responsive UI
- Step-by-step tree updates on insertion

## 🧠 What is AVL Tree?

An AVL Tree is a self-balancing Binary Search Tree where the difference between heights of left and right subtrees (balance factor) is at most 1 for every node.

## 🔧 Tech Stack

- React (Vite or CRA)
- JavaScript (ES6+)
- CSS for styling
- SVG for tree visualization
